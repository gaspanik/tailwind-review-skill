# Hex → Built-in Tailwind Color Reference

Reference for Dimension 3-C of tailwind-review. Read when hardcoded hex / rgb color values are found in arbitrary classes.

Exact match first, then nearest by hex distance. Approximate matches must be flagged for visual confirmation.
